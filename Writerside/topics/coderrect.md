# coderrect

## coderrect 静态代码检查

### 1、install coderrect
```Bash
wget https://coderrect.com/x/coderrect-linux-1.1.3.tar.gz
tar zxf coderrect-linux-1.1.3.tar.gz
export PATH=$PWD/coderrect-linux-1.1.3/bin:$PATH 
```
### 2、install coderrect plug
```Bash
Coderrect Jenkins Plugin
```
